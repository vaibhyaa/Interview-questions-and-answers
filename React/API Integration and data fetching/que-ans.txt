315. What is race condition in data fetching?
A race condition happens when multiple API calls are fired, and responses arrive out of order, causing outdated data to overwrite the latest state



316. How do you prevent race conditions in search requests?
Debouncing (e.g., lodash debounce)
Cancel previous requests (AbortController)
React Query’s queryKey change handling
